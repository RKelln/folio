# Grant Drafting

How to draft grant application text using {org_name}'s historical archive as precedent. Ground every claim in real organizational data.

## Before you write — search first

Always search the archive before drafting:

1. Identify the funder and question (e.g. "OAC Community Engagement section")
2. Find past answers with the two-tool pipeline:
   - `sage-wiki search` → find relevant documents
{?agentmap_enabled}   - `agentmap search` → find the section heading within those docs
{/agentmap_enabled}
3. Present past answers organized by year
4. Draft the new answer grounded in precedent

## Drafting principles

### 1. Ground everything in source documents

Every claim should be traceable to a specific document. Use exact numbers and language from the archive.

### 2. Match funder tone

Each funder has distinct conventions. Check `wiki/AGENTS.md` for funder-specific tone guidance.

### 3. Cite your sources

Note which documents provided which information. This lets the human reviewer verify facts.

## Output format

```
## [Section name]

[Drafted text]

### Sources
- [filename.md]: [what was used]
- ...

### Key facts used
- [specific claim] — from [filename.md]
```
